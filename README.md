# CO2-Emissions-Prediction-Model
# CO2 Emissions Prediction 🚗💨

A machine learning project that predicts CO2 emissions of vehicles based on 
features like engine size, cylinders, and fuel consumption using Linear Regression.

## Dataset
- Source: Kaggle — CO2 Emissions Canada
- 7000+ vehicle records with features like engine size, fuel consumption, and transmission type

## Features Used
- Engine Size (L)
- Cylinders
- Fuel Consumption City (L/100 km)
- Fuel Consumption Hwy (L/100 km)
- Fuel Consumption Comb (L/100 km)
- Fuel Consumption Comb (mpg)

## Results
| Metric | Score |
|--------|-------|
| MSE    | 359.04 |
| RMSE   | 18.95  |
| R²     | 0.894  |

## Libraries Used
- Pandas
- NumPy
- Matplotlib
- Scikit-learn

## How to Run
1. Clone the repository
2. Upload the dataset to your environment
3. Run the notebook CO2_emission.ipynb
